# SkyBrain 终端页面设计

## 1. 项目概述

- **项目名称**: SkyBrain 无人机管理系统
- **页面**: 终端 (Dashboard)
- **类型**: 数据密集型指挥中心仪表盘
- **目标用户**: 校园安保人员、运维人员、管理员

### 2.2 响应式断点

- **Desktop**: >= 1280px - 4列网格
- **Tablet**: 768px - 1279px - 2列网格
- **Mobile**: < 768px - 1列堆叠

### 2.3 视觉设计

#### 颜色系统
- **主色**: `#0ea5e9` (天蓝色 - 科技感)
- **背景**: `#09090b` (深黑)
- **卡片背景**: `#18181b` (深灰)
- **边框**: `#27272a`
- **文字主色**: `#fafafa`
- **文字次要**: `#a1a1aa`

#### 状态颜色
- **在线/正常**: `#22c55e` (绿色)
- **警告**: `#eab308` (黄色)
- **离线/错误**: `#ef4444` (红色)
- **信息**: `#3b82f6` (蓝色)

#### 字体
- **标题**: Inter Bold, 20px
- **卡片标题**: Inter Semibold, 14px
- **数字**: Inter Bold, 28px
- **标签**: Inter Regular, 12px

- 点击可查看详情或跳转到对应页面

#### 2.4.7 实时监控 Grid
- 3x3 视频卡片布局
- 每个卡片显示：
  - 无人机名称
  - 在线状态指示点
  - 电池电量百分比
- 点击卡片可放大查看
- 支持实时视频流播放

## 3. 功能设计

### 3.1 数据展示
- 实时读取 mock 数据
- 自动计算统计指标
- 支持数据刷新

### 3.2 交互功能
- 视频卡片点击放大
- 告警列表点击跳转
- 卡片悬停显示详情

### 3.3 动画效果
- 数字变化动画
- 卡片渐入动画
- 告警列表滚动

## 4. 验收标准

1. ✅ 页面显示所有6个核心数据卡片
2. ✅ 数据显示正确（从mock数据计算）
3. ✅ 响应式布局正常
4. ✅ 颜色编码正确显示
5. ✅ 视频Grid显示9个无人机视频
6. ✅ 告警列表显示最新5条
7. ✅ 页面整体美观、科技感强

## 5. 技术实现

- **框架**: React + TypeScript
- **UI组件**: shadcn/ui
- **图标**: lucide-react
- **数据源**: 使用现有的 mock 数据
