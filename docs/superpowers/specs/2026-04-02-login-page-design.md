# 登录页面设计文档

**日期：** 2026-04-02
**项目：** SkyBrain 校园无人机巡检系统
**模块：** 登录页面

---

## 1. 需求概述

为 SkyBrain Web 应用创建登录页面，仅包含登录功能，登录成功后跳转到 `/monitor/multi`。

---

## 2. UI/UX 设计

### 2.1 整体风格

- **主题：** 浅色柔和主题
- **配色方案：** 暖白色背景 + 珊瑚橙色主色调
- **布局：** 左右分栏（左侧品牌区 + 右侧登录框）

### 2.2 色彩方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 背景色 | 暖白色 | `#FFFBF5` |
| 登录卡片 | 白色 | `#FFFFFF` |
| 主色调（按钮） | 珊瑚橙 | `#FF7A5C` |
| 主色调悬停 | 深珊瑚橙 | `#FF6A4C` |
| 次要文字 | 灰色 | `#6B7280` |
| 边框色 | 浅灰 | `#E5E7EB` |
| 输入框背景 | 浅灰白 | `#F9FAFB` |

- **左侧品牌区：** 宽度 50%，包含 Logo、标题、副标题
- **右侧登录区：** 宽度 50%，垂直居中登录表单
- **响应式：** 移动端时左右堆叠，登录框在上

### 2.4 组件规格

#### 登录表单
- 用户名输入框（支持中英文）
- 密码输入框（支持显示/隐藏密码切换）
- 记住我复选框
- 登录按钮（全宽）
- "忘记密码？"链接（辅助功能）

#### 表单验证
- 用户名：必填
- 密码：必填，最少 6 位
- 验证触发：输入时即时验证，提交时整体验证
- 错误提示：红色文字，密码输入框下方显示

#### 交互状态
- 按钮 hover：颜色变深，轻微上移
- 输入框 focus：边框变为橙色，带柔和阴影
- 加载状态：按钮显示加载动画，禁用点击

#### 键盘支持
- Enter 键：提交登录表单
- Tab 键：切换输入焦点

#### 错误处理
- 用户名/密码错误：显示"用户名或密码错误"提示
- 网络错误（模拟）：显示"登录失败，请稍后重试"

#### 记住我功能
- 勾选：登录状态保留 7 天
- 未勾选：登录状态保留 1 天（session）

## 3. 功能实现

### 3.1 登录流程

1. 用户输入用户名和密码
2. 点击登录按钮
3. 表单验证通过后，模拟登录（无后端接口）
4. 存储登录状态到 localStorage（模拟 session）
5. 跳转到 `/monitor/multi`

### 3.2 模拟登录数据

```typescript
// 模拟用户数据
const MOCK_USER = {
  username: 'admin',
  password: '123456',
  name: '管理员',
  role: 'admin'
}
```

- 移除 `/register` 路由
- 保留 `/login` 路由
