# 布局组件 (Layout Components)

> 提供应用整体布局结构的组件

---

## 组件清单

| 序号 | 组件名称 | 说明 |
|------|----------|------|
| 1 | `AppLayout` | 主应用布局（侧边栏+顶部+内容） |
| 2 | `AuthLayout` | 登录/注册页面布局 |
| 3 | `BlankLayout` | 空白布局（用于弹窗等） |
| 4 | `DashboardLayout` | 仪表盘专用布局 |
| 5 | `MonitorLayout` | 监控专用布局（全屏） |

---

## 1. AppLayout（主应用布局）

### 用途
主应用的标准布局框架

### 结构
```
┌──────────────────────────────────────────────────────────────┐
│  Header (固定高度 64px)                                      │
│  ┌────────────────────────────────────────────────────────┐  │
│  │ Logo | 标题 | 搜索 | 通知 | 用户菜单                   │  │
│  └────────────────────────────────────────────────────────┘  │
├────────────┬─────────────────────────────────────────────────┤
│  Sider     │  Content                                        │
│  (可折叠)  │  ┌────────────────────────────────────────────┐ │
│            │  │ Breadcrumb                                 │ │
│  ┌──────┐  │  ├────────────────────────────────────────────┤ │
│  │ Menu │  │  │                                            │ │
│  │      │  │  │         页面主要内容                       │ │
│  │      │  │  │                                            │ │
│  │      │  │  │                                            │ │
│  │      │  │  │                                            │ │
│  └──────┘  │  └────────────────────────────────────────────┘ │
└────────────┴─────────────────────────────────────────────────┘
```

### Props
```typescript
interface AppLayoutProps {
  children: React.ReactNode;
  collapsed?: boolean;        // 侧边栏是否折叠
  onCollapse?: (collapsed: boolean) => void;
}
```

## 2. AuthLayout（认证布局）

### 用途
登录、注册等认证页面专用布局

### 结构
```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│                      ┌──────────────┐                       │
│                      │              │                       │
│                      │   登录表单   │                       │
│                      │              │                       │
│                      └──────────────┘                       │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### Props
```typescript
interface AuthLayoutProps {
  children: React.ReactNode;
}
```

## 3. BlankLayout（空白布局）

### 用途

interface BlankLayoutProps {

## 4. DashboardLayout（仪表盘布局）

### 用途
数据大屏、仪表盘专用全屏布局

### 结构
```
┌──────────────────────────────────────────────────────────────┐
│  标题栏 (可选)                                               │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│                    大屏内容区域                              │
│                    (全屏/高分屏适配)                         │
│                                                              │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### Props
```typescript
interface DashboardLayoutProps {
  title?: string;
  showHeader?: boolean;
  children: React.ReactNode;
}
```

---

## 5. MonitorLayout（监控布局）

### 用途
视频监控、实时监控专用布局

### 结构
```
┌──────────────────────────────────────────────────────────────┐
│  顶部栏 (可选: 标题 + 无人机选择 + 控制按钮)                 │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│                                                              │
│                    视频/监控内容                            │
│                    (16:9 或自适应)                          │
│                                                              │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│  底部信息栏 (可选: 无人机状态 + 控制条)                      │
└──────────────────────────────────────────────────────────────┘
```

### Props
```typescript
interface MonitorLayoutProps {
  showHeader?: boolean;
  showFooter?: boolean;
  children: React.ReactNode;
}
```
