# 设施巡检模块 (Facility Inspection)

> 提供建筑巡检、设施检查、施工监控、缺陷管理功能

---

## 页面清单

| 序号 | 页面名称 | 路由 | 描述 |
|------|----------|------|------|
| 1 | 建筑巡检 | `/inspection/building` | 建筑巡检任务与报告 |
| 2 | 设施检查 | `/inspection/infrastructure` | 电力、管网等设施检查 |
| 3 | 施工监控 | `/inspection/construction` | 施工进度与安全监控 |
| 4 | 巡检报告 | `/inspection/reports` | 巡检报告查看 |
| 5 | 缺陷管理 | `/inspection/defects` | 巡检发现缺陷管理 |

---

## 页面1：建筑巡检 (Building Inspection)

### 路由
```
/inspection/building
```

### 页面布局
```
┌─────────────────────────────────────────────────────────────────┐
│  顶部：标题 + 新建巡检按钮 + 筛选                               │
├─────────────────────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────────────────────────┐   │
│  │  建筑列表 (BuildingList)                                  │   │
│  │                                                          │   │
│  │  ┌──────────┐  ┌──────────┐  ┌──────────┐               │   │
│  │  │ 教学楼A  │  │ 教学楼B  │  │ 体育馆   │               │   │
│  │  │ 上次:昨天│  │ 上次:3天前│ │ 上次:1周前│               │   │
│  │  │ [巡检]   │  │ [巡检]   │  │ [巡检]   │               │   │
│  │  └──────────┘  └──────────┘  └──────────┘               │   │
│  └──────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────┤
│  巡检任务列表 (DataTable)                                       │
│  建筑 | 巡检类型 | 状态 | 上次巡检 | 缺陷数 | 操作              │
├─────────────────────────────────────────────────────────────────┤
│  巡检类型筛选: [外墙] [屋顶] [高空抛物] [保温] [玻璃]          │
└─────────────────────────────────────────────────────────────────┘
```

### 包含组件
- `CreateInspectionButton`（新建巡检按钮）
- `BuildingList`（建筑列表）
- `BuildingCard`（建筑卡片）
- `InspectionButton`（执行巡检按钮）
- `InspectionTypeFilter`（巡检类型筛选）
- `DataTable`（巡检任务表）
- `LastInspectionInfo`（上次巡检信息）
- `DefectCountBadge`（缺陷数量徽章）

### 功能描述
1. 建筑列表展示
2. 按建筑执行巡检任务
3. 巡检类型选择（外墙、屋顶、高空抛物等）
4. 巡检历史查询
