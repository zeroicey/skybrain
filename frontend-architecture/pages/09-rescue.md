# 搜索救援模块 (Search & Rescue)

> 提供物品查找、人员搜索、搜救记录等功能

---

## 页面清单

| 序号 | 页面名称 | 路由 | 描述 |
|------|----------|------|------|
| 1 | 物品查找 | `/rescue/items` | 丢失物品查找任务 |
| 2 | 人员搜索 | `/rescue/people` | 走失人员搜寻任务 |
| 3 | 搜救记录 | `/rescue/records` | 搜救历史记录 |
| 4 | 搜救详情 | `/rescue/:id` | 搜救任务详情 |

---

## 页面1：物品查找 (Item Search)

### 路由
```
/rescue/items
```

### 页面布局
```
┌─────────────────────────────────────────────────────────────────┐
│  顶部：标题 + 新建查找任务                                      │
├─────────────────────────────────────────────────────────────────┤
│  查找任务列表                                                   │
│  ┌────────────────────────────────────────────────────────┐     │
│  │ 🎒 黑色背包 - 操场区域                                   │     │
│  │    丢失时间: 14:00 | 状态: 寻找中                       │     │
│  │    发布人: 张三 | 悬赏: 50元                            │     │
│  │    [查看位置] [完成]                                     │     │
│  ├────────────────────────────────────────────────────────┤     │
│  │ 📱 手机 - 图书馆                                         │     │
│  │    ...                                                   │     │
│  └────────────────────────────────────────────────────────┘     │
└─────────────────────────────────────────────────────────────────┘
```

### 包含组件
- `CreateSearchTaskButton`（新建查找任务）
- `SearchTaskList`（查找任务列表）
- `ItemSearchCard`（物品查找卡片）
- `ItemTypeIcon`（物品类型图标）
- `LocationInfo`（位置信息）
- `RewardBadge`（悬赏徽章）
- `ViewLocationButton`（查看位置）
- `CompleteButton`（完成按钮）

### 功能描述
1. 丢失物品查找任务列表
2. 新建查找任务
3. 物品位置定位
4. 任务完成标记

## 页面2：人员搜索 (People Search)

### 路由
```
/rescue/people
```

### 页面布局
```
┌─────────────────────────────────────────────────────────────────┐
│  顶部：标题 + 新建搜救任务                                      │
├─────────────────────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────────────────────────┐   │
│  │  当前搜救任务                                            │   │
│  │                                                          │   │
│  │    目标: 走失老人（王建国，72岁）                        │   │
│  │    特征: 身高170cm, 身穿蓝色外套                         │   │
│  │    走失时间: 14:00 走失地点: 校园东门                   │   │
│  │                                                          │   │
│  │    搜救地图                                              │   │
│  │    (MapView + 热成像 + 搜索范围)                        │   │
│  │                                                          │   │
│  │    状态: 搜索中 | 参与无人机: 3架                        │   │
│  │    [停止搜救] [添加无人机] [通知家属]                    │   │
│  └──────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────┤
│  历史搜救任务                                                   │
│  目标 | 类型 | 状态 | 时间 | 结果                              │
└─────────────────────────────────────────────────────────────────┘
```

### 包含组件
- `CreateRescueTaskButton`（新建搜救任务）
- `CurrentRescuePanel`（当前搜救面板）
- `TargetInfo`（目标信息）
- `FeatureDescription`（特征描述）
- `MissingTime`（走失时间）
- `RescueMap`（搜救地图）
- `ThermalOverlay`（热成像覆盖层）
- `SearchRange`（搜索范围）
- `RescueStatus`（搜救状态）
- `DroneCount`（参与无人机数）
- `StopRescueButton`（停止搜救）
- `AddDroneButton`（添加无人机）
- `NotifyFamilyButton`（通知家属）
- `HistoryTaskList`（历史任务列表）

### 功能描述
1. 人员搜救任务管理
2. 目标信息展示
3. 热成像搜索
4. 搜索范围可视化
5. 多无人机协同搜索
6. 家属通知

## 页面4：搜救详情 (Rescue Detail)

### 路由
```
/rescue/:id
```

### 页面布局
```
┌─────────────────────────────────────────────────────────────────┐
│  顶部：← 返回 | 搜救任务详情 | 状态标签                         │
├─────────────────────────────────────────────────────────────────┤
│  ┌──────────────────────────┐ ┌────────────────────────────┐   │
│  │                          │ │    任务信息                │   │
│  │    任务地图              │ │                            │   │
│  │    (MapView + 轨迹)      │ │  - 任务类型: 人员搜救     │   │
│  │                          │ │  - 目标: 王建国           │   │
│  │                          │ │  - 走失时间: 14:00        │   │
│  └──────────────────────────┘ │  - 搜索时长: 2h 30m       │   │
│                               │  - 参与设备: 3架          │   │
│  ┌──────────────────────────┐ │                            │   │
│  │    实时视频              │ │    参与设备                │   │
│  │    (VideoPlayer)         │ │    无人机1 | 无人机2 | 无人机3    │
│  │                          │ │                            │   │
│  └──────────────────────────┘ └────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────┤
│  任务日志 Timeline                                              │
└─────────────────────────────────────────────────────────────────┘
```

### 包含组件
- `GoBack`（返回）
- `TaskMap`（任务地图）
- `SearchTrack`（搜索轨迹）
- `LiveVideo`（实时视频）
- `TaskInfoPanel`（任务信息）
- `DeviceList`（设备列表）
- `TaskLog`（任务日志）

### 功能描述
1. 搜救任务详情
2. 搜索轨迹展示
3. 实时视频回传
4. 任务日志
