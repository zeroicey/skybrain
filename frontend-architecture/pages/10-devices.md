# 设备管理模块 (Device Management)

> 提供无人机、机库、电池、维护记录等设备管理功能

---

## 页面清单

| 序号 | 页面名称 | 路由 | 描述 |
|------|----------|------|------|
| 1 | 无人机列表 | `/devices/drones` | 无人机管理 |
| 2 | 无人机详情 | `/devices/drones/:id` | 无人机详情、状态 |
| 3 | 机库管理 | `/devices/hangars` | 自动机库、充电桩管理 |
| 4 | 电池管理 | `/devices/batteries` | 电池状态、充电管理 |
| 5 | 维护记录 | `/devices/maintenance` | 设备维护历史 |
| 6 | 设备添加 | `/devices/add` | 添加新设备 |

---

## 页面1：无人机列表 (Drone List)

### 路由
```
/devices/drones
```

### 页面布局

```
│  顶部：标题 + 添加无人机 + 批量操作                             │
├─────────────────────────────────────────────────────────────────┤
│  筛选: [状态▼] [型号▼] [位置▼] [搜索名称]                      │
├─────────────────────────────────────────────────────────────────┤
│  无人机网格                                                    │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐           │
│  │ 🛸 无人机1│ │ 🛸 无人机2│ │ 🛸 无人机3│ │ 🛸 无人机4│           │
│  │ 状态:在线 │ │ 状态:飞行 │ │ 状态:离线 │ │ 状态:充电 │           │
│  │ 电池:85% │ │ 电池:72% │ │ 电池:0%  │ │ 电池:95% │           │
│  │ 今日:2h  │ │ 今日:1h  │ │ 今日:0h  │ │ 今日:0h  │           │
│  │ [详情]   │ │ [详情]   │ │ [详情]   │ │ [详情]   │           │

│  底部：统计（在线: 5 / 总数: 8） + 分页器                       │
└─────────────────────────────────────────────────────────────────┘
```

### 包含组件
- `AddDroneButton`（添加无人机）
- `BatchActionBar`（批量操作）
- `FilterForm`（筛选表单）
- `DroneStatusSelect`（状态筛选）
- `DroneModelSelect`（型号筛选）
- `DroneGrid`（无人机网格）
- `DroneCard`（无人机卡片）
- `DroneStatus`（状态指示）
- `BatteryIndicator`（电池电量）
- `FlightTimeToday`（今日飞行时长）

### 功能描述
1. 无人机列表/网格展示
2. 按状态、型号筛选
3. 无人机详情查看
4. 批量操作（批量开关机）
