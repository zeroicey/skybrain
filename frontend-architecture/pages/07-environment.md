# 环境监测模块 (Environment Monitoring)

> 提供空气质量、校园环境、污染溯源等监测功能

---

## 页面清单

| 序号 | 页面名称 | 路由 | 描述 |
|------|----------|------|------|
| 1 | 空气质量 | `/environment/air` | PM2.5/PM10、气体检测数据 |
| 2 | 校园环境 | `/environment/campus` | 绿化、水体、噪音、温度 |
| 3 | 监测报表 | `/environment/reports` | 环境数据报表 |
| 4 | 污染溯源 | `/environment/pollution` | 污染源定位分析 |

---

## 页面1：空气质量 (Air Quality)

### 路由
```
/environment/air
```

### 页面布局
```
┌─────────────────────────────────────────────────────────────────┐
│  顶部：标题 + 监测点选择 + 时间范围                             │
├─────────────────────────────────────────────────────────────────┤
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐               │
│  │ PM2.5   │ │ PM10    │ │ CO₂     │ │ SO₂     │               │
│  │ 35μg/m³ │ │ 55μg/m³ │ │ 420ppm  │ │ 8ppb   │               │
│  │ 良 ✓    │ │ 良 ✓    │ │ 优 ✓    │ │ 优 ✓    │               │
│  └─────────┘ └─────────┘ └─────────┘ └─────────┘               │
├─────────────────────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────────────────────────┐   │
│  │    空气质量地图                                          │   │
│  │    (MapView + 监测点标记 + 颜色分级)                     │   │
│  │                                                          │   │
│  │    🟢 良  🟡 轻度  🔴 中度  ⚫ 重度                      │   │
│  └──────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────┤
│  ┌──────────────────────────┐ ┌────────────────────────────┐   │
│  │  24小时趋势               │ │  各监测点对比             │   │
│  │  (LineChart)              │ │  (BarChart)               │   │
│  └──────────────────────────┘ └────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
```

### 包含组件
- `MonitoringPointSelect`（监测点选择）
- `TimeRangePicker`（时间范围）
- `AQIIndicator`（空气质量指数指示器）
- `PM25Card`（PM2.5卡片）
- `PM10Card`（PM10卡片）
- `CO2Card`（二氧化碳卡片）
- `SO2Card`（二氧化硫卡片）
- `MapView`（监测地图）
- `MonitorMarker`（监测点标记）
- `AQIColorMarker`（颜色分级标记）
- `LineChart`（24小时趋势）
- `BarChart`（监测点对比）

### 功能描述
1. 空气质量指数展示
2. 监测点分布地图
3. 24小时趋势曲线
4. 多监测点对比

2. 绿化覆盖率统计

4. 噪音水平监测

## 页面4：污染溯源 (Pollution Source)

### 路由
```
/environment/pollution
```

### 页面布局
```
┌─────────────────────────────────────────────────────────────────┐
│  顶部：标题 + 污染事件选择 + 分析按钮                           │
├─────────────────────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────────────────────────┐   │
│  │  污染溯源地图                                            │   │
│  │  (MapView + 扩散模型 + 源点标记)                         │   │
│  │                                                          │   │
│  │        ○ ← 污染源                                        │   │
│  │       ╱ ╲                                               │   │
│  │      ○   ○  扩散范围                                    │   │
│  │                                                          │   │
│  └──────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────┤
│  溯源分析结果                                                   │
│  ┌────────────────────────────────────────────────────────┐     │
│  │  污染类型: 扬尘                                         │     │
│  │  污染源位置: 校园西北角 (建筑工地)                       │     │
│  │  扩散方向: 东南风, 速度 2m/s                            │     │
│  │  影响范围: 约 500m 半径                                 │     │
│  │  建议措施: 建议工地增加洒水频次                          │     │
│  └────────────────────────────────────────────────────────┘     │
├─────────────────────────────────────────────────────────────────┤
│  历史溯源记录                                                   │
│  时间 | 污染类型 | 定位结果 | 处理情况                         │
└─────────────────────────────────────────────────────────────────┘
```

### 包含组件
- `PollutionEventSelect`（污染事件选择）
- `AnalyzeButton`（分析按钮）
- `MapView`（溯源地图）
- `DiffusionModel`（扩散模型）
- `SourceMarker`（源点标记）
- `AnalysisResult`（分析结果）
- `PollutionType`（污染类型）
- `SourceLocation`（源点位置）
- `DiffusionDirection`（扩散方向）
- `InfluenceRange`（影响范围）
- `SuggestionPanel`（建议措施）
- `HistoryRecord`（历史记录）

### 功能描述
1. 污染源定位
2. 扩散模型可视化
3. 影响范围分析
4. 处理建议
