## 数据流说明

### 统计数据来源
```typescript
// Dashboard 统计数据接口
interface DashboardStats {
  // 无人机统计
  drones: {
    total: number;         // 总数
    online: number;        // 在线
    flying: number;        // 飞行中
    charging: number;      // 充电中
    idle: number;          // 待机
    offline: number;       // 离线
  };

  // 任务统计
  tasks: {
    running: number;       // 执行中
    completed: number;     // 已完成
    failed: number;        // 失败
    completionRate: number; // 完成率
  };

  // 告警统计
  alerts: {
    pending: number;       // 待处理
    critical: number;      // 严重
    warning: number;       // 警告
    info: number;          // 信息
  };

  // 飞行统计
  flight: {
    todayMinutes: number;  // 今日飞行时长
    todayKm: number;       // 今日飞行公里
    monthMinutes: number;  // 本月飞行时长
  };
}
```

### 实时数据更新
- 无人机位置：WebSocket 推送，每5秒更新
- 任务状态：WebSocket 推送，状态变化时更新
- 告警：WebSocket 推送，实时告警弹窗 + 列表更新
- 统计图表：定时轮询，每30秒刷新
