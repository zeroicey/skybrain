# 实时监控模块 (Monitor)

> 提供无人机实时视频监控、录像回放功能

---

## 页面清单

| 序号 | 页面名称 | 路由 | 描述 |
|------|----------|------|------|
| 1 | 视频监控 | `/monitor/live` | 无人机实时图传、视频播放 |
| 2 | 监控回放 | `/monitor/playback` | 历史录像回放、下载 |
| 3 | 多路监控 | `/monitor/multi` | 多无人机同时监控网格视图 |
| 4 | 监控详情 | `/monitor/:id` | 单个无人机监控详情 |

---

## 页面1：视频监控 (Live Monitor)

### 路由
```
/monitor/live
```

### 页面布局
```
┌─────────────────────────────────────────────────────────────────┐
│  顶部：标题 + 无人机选择下拉框 + 画质切换 + 全屏按钮            │
├─────────────────────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────────────────────────┐   │
│  │                                                          │   │
│  │                                                          │   │
│  │                    视频播放区域                          │   │
│  │                    (VideoPlayer)                         │   │
│  │                                                          │   │
│  │                                                          │   │
│  │                                                          │   │
│  └──────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────┐    │
│  │  播放控制: ◀◀ | ▶/⏸ | ▶▶ | 🔊 | ⏱00:00/00:00 | ⛶ 全屏 │    │
│  └─────────────────────────────────────────────────────────┘    │
├─────────────────────────────────────────────────────────────────┤
│  底部：无人机信息面板                                           │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐           │
│  │ 高度:50m │ │ 速度:12m/s│ │ 电池:85% │ │ 位置:xxx │           │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘           │
└─────────────────────────────────────────────────────────────────┘
```

### 包含组件
- `VideoPlayer`（视频播放器，支持 HLS/RTMP/FLV）
- `DroneSelector`（无人机选择下拉）
- `QualitySwitcher`（画质切换：流畅/高清/4K）
- `VideoControls`（播放控制条）
- `VolumeControl`（音量控制）
- `Timeline`（播放进度条）
- `FullscreenButton`（全屏按钮）
- `DroneInfoPanel`（无人机信息面板）
- `SnapshotButton`（截图按钮）
- `RecordButton`（录像按钮）

### 功能描述
1. 选择无人机进行实时视频监控
2. 支持多种视频流协议（HLS、RTMP、FLV）
3. 播放控制（播放/暂停/快进/快退）
4. 音量调节和静音
5. 视频截图功能
6. 视频录制功能
7. 全屏播放
8. 实时显示无人机状态信息

### 包含组件
- `DroneSelector`（无人机选择）
- `DatePicker`（日期选择）
- `TimePicker`（时间范围选择）
- `VideoPlayer`（录像播放器）
- `VideoTimeline`（录像时间轴，可拖拽）
- `DataTable`（录像文件列表）
- `DownloadButton`（下载按钮）
- `ExportButton`（批量导出）
- `Pagination`（分页器）

### 功能描述
1. 按无人机和日期筛选录像
2. 录像时间轴显示可回放时间段
3. 录像列表展示历史录像
4. 录像播放控制
5. 单个录像下载
6. 批量导出录像

### 包含组件
- `LayoutSwitcher`（布局切换：1x2/2x2/3x3/自定义）

- `VideoPlayer` × N（多个视频播放器）
- `QuickAddDropdown`（快速添加监控点）
- `FullscreenAllButton`（全部全屏按钮）
- `DroneMiniInfo`（无人机简要信息）

### 功能描述
1. 同时监控多个无人机视频
2. 支持多种布局模式
3. 单个视频全屏查看
4. 快速添加常用监控点
5. 实时显示各无人机状态

## 视频流协议支持

| 协议 | 场景 | 延迟 |
|------|------|------|
| HLS | 通用场景 | 2-5秒 |
| RTMP | 实时监控 | < 1秒 |
| FLV | Web实时 | < 1秒 |
| WebRTC | 超低延迟 | < 300ms |
