# 系统设置模块 (System Settings)

> 提供用户管理、角色权限、系统配置等功能

---

## 页面清单

| 序号 | 页面名称 | 路由 | 描述 |
|------|----------|------|------|
| 1 | 用户管理 | `/settings/users` | 用户列表、添加、编辑 |
| 2 | 角色权限 | `/settings/roles` | 角色与权限管理 |
| 3 | 操作日志 | `/settings/logs` | 系统操作日志 |
| 4 | 系统配置 | `/settings/config` | 系统参数配置 |
| 5 | 通知设置 | `/settings/notification` | 通知渠道配置 |

---

## 页面1：用户管理 (User Management)

### 路由
```
/settings/users
```

### 页面布局
```
┌─────────────────────────────────────────────────────────────────┐
│  顶部：标题 + 添加用户 + 批量导入                               │
├─────────────────────────────────────────────────────────────────┤
│  筛选: [角色▼] [状态▼] [部门▼] [搜索姓名/账号]                │
├─────────────────────────────────────────────────────────────────┤
│  用户列表 (DataTable)                                          │
│  ┌────┬──────────┬────────┬────────┬────────┬────────┬──────┐   │
│  │ #  │ 账号     │ 姓名   │ 角色   │ 部门   │ 状态   │ 操作 │   │
│  ├────┼──────────┼────────┼────────┼────────┼────────┼──────┤   │
│  │ 1  │ admin    │ 张三   │ 管理员 │ 信息中心│ 启用   │ ⋯   │   │
│  │ 2  │ security │ 李四   │ 安保   │ 安保部 │ 启用   │ ⋯   │   │

│  底部：分页器 + 统计（共计 50 用户）                           │
└─────────────────────────────────────────────────────────────────┘

### 包含组件
- `AddUserButton`（添加用户）
- `BatchImportButton`（批量导入）
- `UserFilter`（用户筛选）
- `RoleSelect`（角色筛选）
- `StatusSelect`（状态筛选）
- `DepartmentSelect`（部门筛选）
- `SearchInput`（搜索）
- `DataTable`（用户表格）
- `UserAvatar`（用户头像）
- `UserStatus`（用户状态）
- `EditUserButton`（编辑）
- `DeleteUserButton`（删除）
- `ResetPasswordButton`（重置密码）
- `Pagination`（分页器）

### 功能描述
1. 用户列表展示
2. 添加/编辑/删除用户
3. 用户筛选和搜索

5. 批量导入用户
