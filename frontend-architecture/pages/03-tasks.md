# 任务管理模块 (Task Management)

> 提供任务创建、任务列表、任务调度、定时任务等功能

---

## 页面清单

| 序号 | 页面名称 | 路由 | 描述 |
|------|----------|------|------|
| 1 | 任务列表 | `/tasks` | 所有任务列表、筛选、搜索 |
| 2 | 任务详情 | `/tasks/:id` | 任务详情、执行状态 |
| 3 | 任务创建 | `/tasks/create` | 新建任务向导 |
| 4 | 任务编辑 | `/tasks/:id/edit` | 编辑任务 |
| 5 | 定时任务 | `/tasks/schedule` | 定时任务管理 |
| 6 | 任务日志 | `/tasks/logs` | 任务执行日志 |

---

## 页面1：任务列表 (Task List)

### 路由
```
/tasks
```

### 页面布局
```
┌─────────────────────────────────────────────────────────────────┐
│  顶部：标题 + 新建任务按钮 + 导入/导出                           │

│  │  筛选表单:                                                 │   │

│  └──────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────┤
│  任务列表 (DataTable)                                           │
│  ┌────┬──────────┬────────┬────────┬───────┬────────┬──────┐   │
│  │ #  │ 任务名称 │ 类型   │ 状态   │ 无人机│ 执行人 │ 操作 │   │

│  │ 1  │ 日常巡逻 │ 巡逻   │ 执行中 │ 无人机1│ 系统   │ ⋯   │   │

├─────────────────────────────────────────────────────────────────┤
│  底部：批量操作 + 分页器（显示 1-10，共 100 条）                │
└─────────────────────────────────────────────────────────────────┘
```

### 包含组件
- `SearchForm`（搜索表单）
- `FilterForm`（筛选表单）
- `TaskTypeSelect`（任务类型下拉：巡逻/巡检/保障/配送/搜救/自定义）
- `TaskStatusSelect`（任务状态下拉：待执行/执行中/已完成/失败/已取消）
- `DroneSelect`（无人机选择）
- `DateRangePicker`（时间范围选择）
- `DataTable`（任务列表表格）
- `CreateTaskButton`（新建任务按钮）
- `ImportButton`（导入按钮）
- `ExportButton`（导出按钮）
- `BatchActionBar`（批量操作栏）
- `Pagination`（分页器）

### 功能描述
1. 任务列表展示，支持分页
2. 多维度筛选（任务类型、状态、无人机、时间）
3. 关键字搜索
4. 任务状态可视化（颜色标签）
5. 批量操作（批量删除、批量执行）
6. 导入/导出任务配置

## 页面3：任务创建 (Task Create)

### 路由
```
/tasks/create
```

### 页面布局
```
┌─────────────────────────────────────────────────────────────────┐
│  顶部：步骤指示器                                               │
│  ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐     │
│  │ 1 基本  │───▶│ 2 航线  │───▶│ 3 参数  │───▶│ 4 确认  │     │
│  │ 信息    │    │ 选择    │    │ 设置    │    │ 提交    │     │
│  └─────────┘    └─────────┘    └─────────┘    └─────────┘     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │                                                          │   │
│  │                   步骤内容区域                           │   │
│  │                                                          │   │
│  │   (根据当前步骤显示不同的表单内容)                        │   │
│  │                                                          │   │
│  └──────────────────────────────────────────────────────────┘   │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│  底部：[上一步]                        [下一步/提交任务]        │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤1：基本信息
```
任务名称: [________________________]
任务类型: [巡逻▼] [巡检▼] [保障▼] [配送▼] [搜救▼] [自定义▼]
优先级:   ○ 低  ● 普通  ○ 高  ○ 紧急
任务描述: [________________________]
```

### 步骤2：航线选择
```
选择航线:
┌──────────────────────────────────────────┐
│  ○  校园周界巡逻                         │
│  ○  教学楼巡检                           │
│  ○  宿舍楼巡检                           │
│  ○  自定义航线 [选择/新建]               │
└──────────────────────────────────────────┘

航线预览: [MapView 显示所选航线]
```

### 步骤3：参数设置
```
执行设置:
  无人机: [无人机1▼] [无人机2▼] [+添加更多]
  执行时间: ○ 立即执行  ○ 定时执行 [14:30]
  重复: ○ 不重复  ○ 每日  ○ 每周  ○ 自定义

AI 设置:
  ☑ 开启实时分析
  ☑ 异常自动告警
  ☑ 发现异常时通知: [手机▼] [邮件☐]
```

### 步骤4：确认提交
```
任务确认:
  任务名称: 日常巡逻
  任务类型: 巡逻
  航线: 校园周界巡逻
  执行无人机: 无人机1
  执行时间: 立即执行
  优先级: 普通
  AI分析: 开启

[返回修改]                              [提交任务]
```

### 包含组件
- `Steps`（步骤指示器）
- `TaskForm`（任务表单）
- `RouteSelect`（航线选择器）
- `RoutePreview`（航线预览）
- `MapView`（地图）
- `DroneMultiSelect`（多无人机选择）
- `DateTimePicker`（日期时间选择）
- `RepeatConfig`（重复配置）
- `AISettings`（AI设置）
- `TaskConfirm`（任务确认）
- `SubmitButton`（提交按钮）

### 功能描述
1. 分步任务创建向导
2. 航线选择和预览
3. 执行参数配置
4. AI分析选项
5. 任务确认和提交
